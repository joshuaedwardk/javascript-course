üÜò The "Emergency" Section

Goal
Command
Result
Start Over

Goal
Command
Description
Push to Remote
git push -u origin [branch-name]
Uploads your new branch to GitHub and links it (the -u sets it up for future pushes).
Merge to Main
git merge [branch-name]
Run this while on main to pull changes from your feature branch into it.
Nuke Local
git branch -D [branch-name]
Force Delete: Deletes the branch on your computer even if it hasn't been merged.
Nuke Remote
git push origin --delete [branch-name]
Deletes the branch on GitHub permanently.

üê≥ Docker Compose: WordPress Workflow

Manage your local WordPress environment using Docker Compose commands.

Step
Command
What it does
1. Start
docker compose up -d
Builds/starts containers in detached mode; creates volumes.
2. Confirm
docker compose ps
Check if WordPress and DB containers are Up.
3. Stop
docker compose down
Stops containers. Database volumes and data persist.
4. Restart
docker compose up -d
Brings the site back exactly as it was.
