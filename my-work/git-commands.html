üÜò The "Emergency" Section

Goal
Command
Result
Start Over
git reset --hard origin/main
Wipes all local changes to match GitHub exactly.
Clean Junk
git clean -fd
Deletes untracked files and folders.
Goal
Command
Description
Push to Remote
git push -u origin [branch-name]
Uploads your new branch to GitHub and links it (the -u sets it up for future pushes).
Merge to Main
git merge [branch-name]
Run this while on main to pull changes from your feature branch into it.
Nuke Local
git branch -D [branch-name]
Force Delete: Deletes the branch on your computer even if it hasn't been merged.
Nuke Remote
git push origin --delete [branch-name]
Deletes the branch on GitHub permanently.
